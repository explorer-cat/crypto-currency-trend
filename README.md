# 업비트 실시간 시세 및 코인정보 플랫폼
> 부제 : coinZZang
>

주요 암호화폐(비트코인,이더리움,리플)들의 실시간 시세를 UPBIT Open API를 통해 확인할 수 있으며, 비트코인의 암호화폐 시장 점유율과 시장 심리상태 등 여러가지 
암호화폐 정보를 제공하는 플랫폼입니다. 국내 1위 거래소인 업비트 공지사항을 감지하여 신규상장 또는 입출금 관련 공지사항을 빠르게 확인 할 수 있습니다.
